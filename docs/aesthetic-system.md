# Aesthetic system

Shared visual and tonal language across the ecosystem.

---

## Feeling

Breathable · warm · quiet · intentional · tactile · human

A place you want **stay in** — not a tool you use efficiently.

---

## Palette direction

| Role | Direction |
|------|-----------|
| **Background** | Deep warm charcoal — not cold blue-black |
| **Foreground** | Soft cream, not pure white |
| **Accent** | Aged gold, vermouth amber, muted wine — not neon |
| **Muted** | Stone, olive-gray, kissaten wood |
| **Borders** | Subtle, low contrast — room to breathe |

Native implements warm gold accent. Drift aligns to the same warmth — less zinc-cold, more evening-light.

---

## Typography

- Generous line height and paragraph spacing
- Restrained caps labels — whisper, don't shout
- Headlines: medium weight, tight but not aggressive
- Body: readable, reflective, unhurried

---

## Layout

- Space over density
- Fewer cards, more pause
- No gamification, streaks, or metric theater
- Metrics exist — but framed as **rhythm**, not performance

---

## Language

| Avoid | Prefer |
|-------|--------|
| Optimize | Restore |
| Dashboard | Room / space |
| Intervention | A quiet notice |
| Anti-drift | When rhythm fades |
| Productivity | Continuity |
| User | Person |
| Leverage | What endures |
| Calibration (cold) | Return · remembering |
