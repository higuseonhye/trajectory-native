## Full post

```
Big shift in what we're building.

Drift isn't only cognitive.

People drift in dead environments — fluorescent offices, scroll loops, sterile routines — while still looking functional.

So trajectory-native / trajectory-drift is evolving toward a trajectory-aware human environment system.

Not "be more productive."
Not habit tracking.

→ Are you steering — or just functioning?
→ What environment would help you feel alive enough to choose?

Shipped today:
· daily steering — "what tiny turn did you make today?"
· environment tags on events (office / nature / dead / restorative)
· environmental drift detection (dead atmospheres, awe deprivation)
· org-reasoning-mvp — decision memory API (3rd repo)

native (personal): github.com/higuseonhye/trajectory-native
drift (teams + agents): github.com/higuseonhye/trajectory-drift
memory (decisions): github.com/higuseonhye/org-reasoning-mvp

#BuildInPublic
```
