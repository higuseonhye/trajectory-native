# Naming

Repository names are under reevaluation. GitHub slugs remain stable until a deliberate migration. **Human names** guide copy, READMEs, and future identity.

---

## Criteria

Names should feel:

- cinematic, human, timeless, sensory
- quiet but emotionally resonant
- elegant without sounding luxurious
- philosophical without sounding academic

Names should **not** sound like:

- SaaS startups
- AI infrastructure tools
- corporate wellness apps
- productivity software

---

## Recommended ecosystem

**Umbrella:** **Drift & Return**

Two movements every human knows: leaving oneself, finding the way back. Not diagnostic. Not defensive. Relational.

| GitHub slug | Human name | One line |
|-------------|------------|----------|
| `trajectory-native` | **Return** | Personal rhythm — subtle turns, memory, aliveness |
| `trajectory-drift` | **Drift** | When collectives lose rhythm — noticed gently |
| `org-reasoning-mvp` | **Continuity** | Decision memory — what we chose to remember |
