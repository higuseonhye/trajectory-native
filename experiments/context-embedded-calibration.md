# Experiment: context-embedded calibration

## Hypothesis

Short back-and-forth **on** a reasoning trace (not in Slack) improves directional alignment and leaves an auditable calibration trail.

## Method (v0.2 MVP)

- `CalibrationReply` anchored via `ContextAnchor`
- Seed threads on `r1`, `watchingSignal`, `t1`
- `isCalibration` flag when shared understanding shifts

## Success criteria

- Users prefer replying on-object vs opening Slack
- Calibrated threads correlate with subsequent obsession/feed updates
