# Environment design

Anti-drift atmospheres — spaces, aesthetics, and sensory conditions that restore aliveness and trajectory energy.

---

## Core principle

**Dead environments create drift.**

Drift emerges from:

- lifeless systems and artificial spaces
- sensory deprivation
- emotional numbness
- disconnected routines
- environments that suppress vitality

Not only from bad decisions.

---

## Environment-sensitive trajectory

Human direction is affected by:

| Dimension | Examples |
|-----------|----------|
| **Space** | Office vs garden vs coastline |
| **Light** | Fluorescent vs natural |
| **Texture & materials** | Synthetic vs wood, stone, fabric |
| **Nature** | Plants, water, sky, scale |
| **Atmosphere** | Warmth, cinematic scale, awe |
| **Social energy** | Sterile networking vs meaningful conversation |
| **Embodiment** | Sedentary numbness vs movement, wind, presence |

---

## Dead vs alive (design lens)

| Dead | Alive |
|------|-------|
| Gray fluorescent survival | Natural light, warmth |
| Scroll loops | Sensory richness |
| Fake networking | Honest conversation |
| Extractive productivity | Meaningful making |
| Flat, sterile UI/spaces | Texture, beauty, coherence |
| No awe | Scale, nature, cinema |

---

## Product implications

**Today (software):**
- UI should feel alive, warm, cinematic — not sterile productivity
- Trajectory events can tag **environment context** (where, what atmosphere)
- Drift detection: dead-loop environments, sensory-deprivation patterns
- Intervention: suggest environmental turns, not only cognitive ones

**Future (physical):**
- Curated spaces, lighting, materials, hospitality
- Greenhouse / Mediterranean / botanical reference energy
- Embodied rituals and spatial design

See [`../product-mapping/environment-state.md`](../product-mapping/environment-state.md).
