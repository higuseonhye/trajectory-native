# Human rhythm

Shared philosophical center for **Drift & Return**.

---

## What changed

We moved from trajectory systems (diagnostic, defensive, analytical) to **human rhythm systems** (warm, embodied, quiet).

The old language — anti-drift, optimization, calibration — remains useful internally where precision matters. Outward-facing copy uses **rhythm, return, continuity, warmth**.

---

## Concepts

| Old frame | New frame |
|-----------|-----------|
| Anti-drift | When rhythm fades |
| Intervention | A quiet notice |
| Optimization | Restoration |
| Tracking | Remembering |
| Dashboard | A room |
| Productivity | Continuity |
| Steering | Subtle turns |
| Environment design | Alive atmospheres |

---

## Three expressions

1. **Return** (native) — personal rhythm
2. **Drift** (this repo's sibling) — collective rhythm
3. **Continuity** (org-reasoning) — memory
