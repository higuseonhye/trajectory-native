# Leverage

Types of leverage that compound trajectory — output that scales beyond linear time input.

---

## Leverage types

| Type | Mechanism | Example |
|------|-----------|---------|
| **AI leverage** | Amplify judgment, memory, coordination — not replace it | Decision memory, drift detection, institutional synthesis |
| **Code** | Software that runs without you | Products, automation, reusable libraries |
| **Media** | Content that distributes while you sleep | Owned audience, evergreen content, distribution channels |
| **Systems** | Processes and infrastructure that persist | Documented workflows, org memory, reusable frameworks |
| **Capital** | Money working independently | Equity, assets, reinvested returns |
| **Delegation** | Others executing with your direction | Teams, agents, contractors with clear propagation |
| **Organizational leverage** | Culture, memory, and coordination that scale | Institutional intelligence, alignment systems |

---

## Anti-patterns (low leverage)

- Busy calendars without asset creation
- Task completion without system building
- AI used for speed alone without judgment compounding
- Networking without ownership outcomes
- Learning without application or distribution
- Productivity aesthetics without leverage accumulation

---

## Diagnostic questions

1. If you stopped working tomorrow, what continues?
2. What scales without your direct time?
3. What did you build that can be reused?
4. Is AI increasing leverage or just accelerating labor?
