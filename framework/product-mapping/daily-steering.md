# Daily steering

**Planned v0.9** — replace rigid task management with directional micro-corrections.

---

## Core prompt

> "What tiny turn did you make today — toward direction you actually chose?"

Not: "What did you complete?"  
Not: "How productive were you?"  
Not: "Did you return to normal?"

---

## What a turn looks like

- one honest conversation
- one thing built or shipped
- one avoidance faced
- one loop closed in reality
- one moment of curiosity recovered

---

## vs trajectory events

| Daily steering | Trajectory events |
|----------------|-------------------|
| Intentional log | Observed reality |
| "I chose this turn" | "This happened" |
| Emotional entry | Behavioral signal |
| 1 prompt/day | Continuous ingestion |

Steering entries may spawn or link to trajectory events.

---

## Drift pairing

If no turn logged + drift signals active → gentle intervention:

> "No turn today. That's data too. What's one small move toward signal you trust?"

---

## UI direction

- Single prompt, not a task list
- Optional mood/vitality tag
- Visual: rudder, orbit, tiny arrow — not checkbox aesthetics
- Character-forward optional layer (exploring)
