# State design

Humans are **state-dependent beings**. Trajectory is embodied.

---

## Core insight

A person in a dead gray office is not the same person — psychologically, nervously, directionally — as a person in a greenhouse near the ocean.

**State** includes:

- nervous system condition
- emotional energy
- aesthetic coherence felt in the body
- aliveness vs numbness
- access to awe and beauty

Different states → different trajectories.

---

## State vs cognition

| Cognition-only model | State-aware model |
|---------------------|-------------------|
| "Think better" | "What state am I in?" |
| "Decide harder" | "What environment shaped this state?" |
| Reflection fixes drift | Environment + state restore signal |
| Mind over body | Embodied navigation |

---

## Drift through state collapse

People can remain **functional** while state collapses:

- emotionally numb but productive
- alive on paper, dead in body
- no awe, no surprise, no vitality
- adapted to dead environments

Product should detect **state drift**, not only cognitive drift.

---

## State restoration (not optimization)

| Restores state | Drains state |
|----------------|--------------|
| Nature, light, plants | Fluorescent isolation |
| Movement, wind, scale | Scroll numbness |
| Beauty, texture | Sterile repetition |
| Warm conversation | Performative contact |
| Cinematic awe | Flat survival |

**Environmental turns** are valid steering — leave the room, walk outside, change light, seek scale.

## Product mapping (planned)

| Capability | Role |
|------------|------|
| State check-in | "What state are you in?" (lightweight, not mood tracking theater) |
| Environment tags on events | Where / atmosphere when drift or turn happened |
| Vitality vs function | Momentum surface — already partial proxy |
| Environmental intervention | "Your loops are indoor/sterile — one environmental turn?" |
