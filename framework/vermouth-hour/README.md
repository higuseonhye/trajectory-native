# Vermouth Hour

A physical-layer north star — not a product name (yet), but an **atmospheric anchor** for everything Drift & Return might become in the world.

---

## What it is

The hour before evening fully arrives. Vermouth, light, conversation, warmth, unhurried presence.

Reference energy:

- old wine bars in Madrid
- vermouth hour culture
- Italian aperitivo
- Japanese kissaten
- greenhouse cafés at golden hour
- imperfect but deeply intentional spaces

Not luxury. Not startup culture. **Time embedded in a room.**

---

## Why it matters for software

Return and Drift are quiet rooms in code. Vermouth Hour asks:

> What would it feel like if this philosophy had a door you could walk through?

Software holds memory and rhythm. Physical spaces hold **embodiment** — scent, light, wood, silence, the pace of conversation.

---

## Design principles

| Principle | Meaning |
|-----------|---------|
| **Warmth over efficiency** | No extractive UX; stay awhile |
| **Texture over flatness** | Materials, typography, history in surfaces |
| **Ritual over gamification** | Today's turn, not streaks |
| **Silence over noise** | Pauses are features |
| **Gathering over networking** | Alive conversation, not performance |

---

## Possible expressions (future)

- intimate gatherings with Return / Drift philosophy
- greenhouse or kissaten-inspired spaces
- slow technology installations
- journals, objects, scent, light
- hosted **vermouth hour** — one subtle turn shared aloud

Software remains the memory layer. Vermouth Hour is what **return** might feel like in a body, in a room.
